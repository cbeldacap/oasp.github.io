Getting started on OASP

Get to know what OASP has to offer and start coding in zero time by following these steps.

Get started with...

  • The platform guide

    First things first. Start your way through OASP by reading the OASP4j Platform Guide

  • View the demo

    There is currently one rather technical demo, simulating a few use cases of a restaurant management application

    • oasp sample application

  • Setup the development environment

    The OASP IDE setup guide will guide you how to get your development environment up and running.

  • Get the code

    Once you are able to code applications with OASP support you surely want to take a peek to the technology underneath and be able to customize for your needs.

    The code is hosted on Github, here are the main repositories

    • OASP4j - GitHub public repo

  • Help making OASP even better ;-)

    Remember this is a joint effort and you are welcome to contribute.
