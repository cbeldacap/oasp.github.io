Getting started on OASP4JS

Get to know what OASP4js has to offer and start coding in zero time by following these steps.

Not familiar with JavaScript-Programming or AngularJS?

  • We would recommend you to take our crash JavaScript course.

  • After that you can read the AngularJS tutorial and our training materials.

 

otherwise - get started with...

  • View the demo

    There is currently one rather technical demo, simulating a few use cases of a restaurant management application

    • oasp sample application

  • Setup the development environment

    The OASP4js IDE setup guide will guide you how to get your AngularJs development environment up and running.

  • Get the code

    The code is hosted on Github, here is the OASP4js main repository

  • Read the tutorial

    Read and follow the tutorial at OASP4js Client Tutorial

  • Help making OASP even better ;-)

    Remember this is a joint effort and you are welcome to contribute.
